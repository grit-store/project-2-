# Professional Exteriors

Atlanta's premium exterior & roofing experts website.

## Features
- Gold Edition design aesthetics (Champagne Gold / Obsidian dark theme)
- Infinite intersection scroll-reveal animations
- Lenis Inertial Smooth Scrolling
- Custom Gold Cursor with particle trail physics
- Responsive mobile dropdown menus
- Complete services sections (Roofing, Gutters, Washing, Storm Damage, Gallery)
- Fully functional category filters for project gallery (masonry grid & lightbox modal)
- Complete legal and support pages (Privacy, Terms, Warranty)
